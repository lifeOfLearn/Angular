directives

  • 結構指令: 需要import: CommonModule
    • NgIf
      • 結構性語法: *ngIf
        • 1.ngIfBoolean: {{ ngIfBoolean}}
          1-Show NgIf: Boolean = true
          2-Show NgIf: Boolean = false
        • 2.class: {name:string} | null = {{class}}
          class.name = {{class.name}}
        • 3.ngIf : if else
          ngIfBoolean = true, not show else
          ng-template #falseView : ngIfBoolean = false, show else
      • 非結構性語法,使用屬性語法: [ngIf]
        • 4.if exist then <ng-template> show
          [ngIf]=true class.name = {{class.name}}
      • 第二點的*ngIf 等價 第四點的[ngIf] , 因為抓到*後自動將整個轉成 <ng-template>,另外<ng-template>不會真正被渲染,因為不是DOM元素
    • NgFor
      • 結構性語法: *ngFor
        • 1. *ngFor = "let 迴圈跑的變數 of 要抓取的內容變數"
          • ngForItems.id = {{ item.id }} , ngForItems.name = {{ item.name }}
        • 2.增加index
          • index = {{i}} , ngForItems.id = {{ item.id }} , ngForItems.name = {{ item.name }}
        • 3.奇偶
          • even odd , index = {{i}} , ngForItems.id = {{ item.id }} , ngForItems.name = {{ item.name }}
        • 4.追蹤函數trackBy
          • even odd , index = {{i}} , ngForItems.id = {{ item.id }} , ngForItems.name = {{ item.name }}
      • 非結構性語法,使用屬性語法: ngFor
        • 5.ng-template寫法
          • even odd , index = {{i}} , ngForItems.id = {{ item.id }} , ngForItems.name = {{ item.name }}
      • 第四點的*ngFor 等價 第五點的ngFor
